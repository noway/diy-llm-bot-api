# DIY LLM Bot - API
API counterpart for [github.com/noway/diy-llm-bot](https://github.com/noway/diy-llm-bot).

## Try it out
You can try it out on [diy-llm-bot.com](https://diy-llm-bot.com/).

## Running

In development:

```bash
bun dev
```

In production:

```bash
pm2 start bun --name diy-llm-bot-api -- start
```
